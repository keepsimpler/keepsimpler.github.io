Rational Function

In mathematics, a rational function is any polynomial divided by a polynomial, i.e. both the numerator and denominator are polynomials.

A particular rational function is:

f(x)=ax+cbx+d(1)
whose numerator and denominator are both first-order polynomials, i.e. linear functions.

Function (1) is nearly linear for small values of $x$ but reaches a constant value as $x$ becomes large (an "asymptote").

%matplotlib inline

import numpy as np
import matplotlib.pyplot as plt

# a meta-function
def rational_function(a, b, c, d):
    return lambda x: (a * x + c)/(b * x + d)
x = np.arange(0, 20, 0.1, dtype = float)
f1 = rational_function(1, 1, 0, 1)
# print f1(4.)
fx1 = map(f1, x)
f2 = rational_function(1, 1, 0, 2)
fx2 = map(f2, x)
#pylab.rcParams['figure.figsize'] = (8.0, 4.0)
plt.figure()
plt.plot(x, fx1, 'g-', linewidth = 2, label = "a=1,b=1,c=0,d=1")
plt.plot(x, fx2, 'r-', linewidth = 2, label = "a=1,b=1,c=0,d=2")
plt.xlabel = 'x'
plt.ylabel = 'f'
plt.tight_layout()
plt.legend(loc='lower right')
plt.savefig('naka-rushton.png', bbox_inches="tight")

Function (1) starts at a height $c/d$ (the intercept) when $x = 0$. It rises or falls linearly, at first, with slope $(ad -bc)/d^2$ (the value of $dx/dt$ when $x$ is zero). As $x$ becomes very large, the term $ax$ dominates $c$ in the numerator, and the term $bx$ dominates $d$ in the denominator. Consequently the function eventually reaches an asymptote at $ax/bx = a/b$.

Michaelis–Menten kinetics function in biochemistry is a kind of rational function where $a = Vm, b = 1, c = 0, d = K_M$.

v=d[P]dt=Vm[S]KM+[S]

In community ecology, the functional response in terms of the per capita increasing rate of predator $x$ as a function of abundance of prey $y$ is also a rational function, where $a = 1, b = h, c = 0, d = 1$.

dxdt=y1+hy
$h$ is the handling time of the predator on the prey.

Logistic Function

A logistic function or logistic curve is a common "S" shape (sigmoid curve), with equation:

f(x)=L1+e−k(x−x0)
where:

$x0$ = the x-value of the sigmoid's midpoint,

$L$ = the curve's maximum value, and

$k$ = the steepness of the curve.

For values of $x$ in the range of real numbers from −∞ to +∞, the S-curve shown below is obtained (with the graph of $f$ approaching L as $x$ approaches +∞ and approaching zero as $x$ approaches −∞).

import numpy as np
import matplotlib.pyplot as plt

dashes = ['--', #    : dashed line
          '-', #     : solid line
          '-.', #   : dash-dot line
          ':', #    : dotted line
           '-']

def NR(C,C50,n):
    return (  ( C / C50)**n /( 1 + ( C / C50)**n ))

figure(1)
C = np.arange(0.0,3.0,0.1)
C50 = np.arange(0.5,2.5,0.5)

for i,c50 in enumerate(C50):
    plt.plot(C, NR(C,c50,2), label = "k = " +str(c50), c= 'k',linestyle = dashes[np.mod(i,len(dashes))] )

plt.xlabel = "Contrast"
plt.ylabel = 'gain'
plt.legend(loc='lower right')
plt.savefig('naka-rushton.png', bbox_inches="tight")
